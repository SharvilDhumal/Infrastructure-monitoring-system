# InfraVision - AI Infrastructure Monitoring System

A modern, responsive landing page for an AI-powered public infrastructure monitoring system built with React.js and Tailwind CSS.

## Features

- 🎨 **Modern Design**: Clean, professional UI with Tailwind CSS
- 📱 **Fully Responsive**: Optimized for all device sizes
- ⚡ **Smooth Animations**: Framer Motion powered animations
- 🚀 **Performance**: Optimized React components
- 🎯 **Accessibility**: Semantic HTML and keyboard navigation

## Sections

- **Navigation**: Fixed header with smooth scrolling navigation
- **Hero Section**: Compelling headline with animated dashboard preview
- **Features**: 4 key features with icons and descriptions
- **Objectives**: Project goals and benefits
- **Call-to-Action**: Contact and demo scheduling
- **Footer**: Contact information and links

## Technologies Used

- React 18.2.0
- Tailwind CSS 3.3.5
- Framer Motion 10.16.4
- Lucide React (Icons)
- Create React App

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Customization

The landing page is fully customizable through:

- **Colors**: Modify the color scheme in `tailwind.config.js`
- **Content**: Update text and data in `LandingPage.jsx`
- **Animations**: Adjust Framer Motion animations
- **Styling**: Use Tailwind utility classes for quick styling changes

## Deployment

Build the project for production:

```bash
npm run build
```

The build folder contains the production-ready files that can be deployed to any static hosting service.
